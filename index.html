judoKit iOS SDK

This is the official judo iOS SDK. It incorporates our mobile specific security toolkit, judoShield, with additional modules to enable easy native integration of payments. If you are writing your app in Objective-C, we highly recommend you to use the judoKitObjC port.

***Due to industry-wide security updates, versions below 5.5.1 of this SDK will no longer be supported after 1st Oct 2016. For more information regarding these updates, please read our blog here.***

What is this project for?

judoKit is a framework for integrating easy, fast and secure payments inside your app with judo. It contains an exhaustive in-app payments and security toolkit that makes integration simple and quick.

Requirements

This SDK requires Xcode 7.3 and Swift 2.2.
